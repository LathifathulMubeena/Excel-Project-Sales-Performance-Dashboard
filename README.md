## 📊 Sales Performance Dashboard
Analyzed 30K+ e-commerce sales records to build an interactive Excel dashboard.

**Highlights:**
- Built Pivot Tables & Charts for Sales by Month, Category, State, Channel & Order Status
- Identified top-performing states and sales channels (Amazon, Myntra, Flipkart)
- Tracked order status trends (Delivered, Cancelled, Returned, Refunded)

**Skills:** Excel, Pivot Tables, Data Visualization, Sales Analysis
